# Tutorial to use this workspace
1. Create your own workspace in your home directory. Example: mkdir -p ~/catkin_racecar_ws/src
2. cd ~/catkin_racecar_ws/src
3. git clone https://github.com/ariielgodoy/FS-CV-perception-sim.git .
4. The last dot is important to not create a new folder with the repo name. And you will be good to go.
5. Remember the good practices learned in the "Control y programación de robots" course. (Creating the pkgs in the src folder...)
